## 🔧 주요 기능

- 🔒로그인 / 회원가입
1. /auth 경로에 로그인 / 회원가입 기능을 개발
2. 이메일과 비밀번호의 유효성을 확인 : 조건을 만족해야 제출 버튼이 활성화
3. 로그인 API를 호출하고, 올바른 응답을 받았을 때 루트 경로로 이동
4. 응답으로 받은 토큰은 로컬 스토리지에 저장
